Mahjongg

Mahjongg is a Mahjongg solitaire application for Kindle, inspired by GNOME Mahjongg. It features tile artwork specifically adapted for grayscale e-ink displays.

Download

  • Download Mahjongg Extension (Latest Release)

Features

  • Tile artwork optimized for 16-level grayscale e-ink screens.
  • Uses the "Easy" layout shape for a better experience on small screens.
  • Simple and intuitive touch controls.
  • Lightweight and responsive.

Installation

  1. Download the kindle-mahjongg-extension.zip file.
  2. Unzip it and copy the resulting folder to the /extensions/ directory on your Kindle.
  3. Launch via KUAL -> Kindle Mahjongg -> Launch.
