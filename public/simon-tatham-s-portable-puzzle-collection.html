Simon Tatham's Portable Puzzle Collection

Simon Tatham's Portable Puzzle Collection brings more than 30 classic logic puzzles to jailbroken Kindles, adapted for quick e-ink play during reading breaks or long trips.

Download

  • Download the latest Kindle puzzles release

Features

  • More than 30 puzzle games adapted for Kindle
  • Builds for HF and PW2 Kindle architectures
  • Launcher app for selecting games on-device
  • Launch from a booklet or KUAL menu
  • Fast, efficient C implementation with a native GTK interface

Some games from the original collection were removed because they need color, keyboard, or mouse input.

Requirements

  • Jailbroken Kindle
  • Compatible HF or PW2 Kindle build from the release package
  • KUAL is optional if you use the provided booklet launcher

Installation

  1. Download the latest release from the Kindle port page.
  2. Extract the downloaded archive on your computer.
  3. Copy the extracted files to your Kindle.
  4. Start the launcher from your Kindle library using the provided booklet, or launch it from KUAL.
