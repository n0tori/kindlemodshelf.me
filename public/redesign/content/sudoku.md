# Sudoku for Kindle

> **Sudoku** is a classic 9x9 puzzle game for jailbroken Kindles, built with the Illusion framework.

## Download

  - [Download sudoku.zip](../downloads/sudoku.zip)

## Features

  - Classic 9×9 Sudoku gameplay
  - Multiple difficulty levels
  - Touch to select a cell; use on-screen number buttons
  - Clean interface designed for e-ink
  - Works offline—no internet required

## Requirements

  - Jailbroken Kindle with Mesquite support
  - Illusion framework compatibility

## How to Install

  1. Download [sudoku.zip](../downloads/sudoku.zip)
  2. Unzip the file on your computer (uncompressed folder + **.sh** script)
  3. Place the **.sh script** and the **folder** in your Kindle’s **documents** folder
  4. Disconnect your Kindle
  5. Tap the new book in your Kindle library to launch

## How to Play

  - Tap a cell to select it
  - Use the on-screen number buttons to fill or change a digit
  - Complete the grid so each row, column, and 3×3 box has 1–9 with no repeats

Challenge your mind with Sudoku on your Kindle!
