# Aisleriot

> **Aisleriot** brings the classic Klondike Solitaire experience to Kindle. It features high-contrast card symbols and an interface specifically optimized for the e-ink screen.

## Download

  - [Download Aisleriot Extension (Latest Release)](https://github.com/jckhng/kindle-aisleriot/releases/latest)

## Features

  - Classic Klondike Solitaire rules.
  - High-contrast card designs and suit symbols for legibility.
  - Touch-friendly layout for easy card movement.
  - Optimized for the Kindle's refresh rate.

## Installation

  1. Download the `kindle-aisleriot-extension.zip` file.
  2. Unzip it and copy the resulting folder to the `/extensions/` directory on your Kindle.
  3. Launch via **KUAL - > Kindle Aisleriot -> Launch**.
