# Kindle BT Keepalive

> A background utility that keeps Bluetooth connections active on Kindle so your headphones stay connected while you read.

## Download

  - [Download Kindle-bt-keepalive](https://github.com/imanubdesigner/kindle-bt-keepalive/releases/)
  - [View Repository](https://github.com/imanubdesigner/kindle-bt-keepalive)

## Features

  - Prevents the Kindle from putting the Bluetooth stack to sleep during long reading intervals
  - Keeps headphones or speakers connected without manual reconnection
  - Lightweight: minimal impact on battery life

## Prerequisites

  - Jailbroken Kindle
  - Bluetooth-capable Kindle model
  - KUAL or SSH access for installation

## Installation

Deploy the script to your extensions folder or run it via terminal. See the [README in the repository](https://github.com/imanubdesigner/kindle-bt-keepalive) for detailed steps.
