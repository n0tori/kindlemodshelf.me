# Draughts

> **Draughts** (Checkers) for Kindle supports both English 8x8 and International 10x10 variants. It features a minimax AI and high-contrast pieces optimized for e-ink legibility.

## Download

  - [Download Draughts Extension (Latest Release)](https://github.com/jckhng/kindle-draughts/releases/latest)

## Features

  - English (8x8) and International (10x10) variants.
  - Minimax AI for challenging computer opponents.
  - High-contrast pieces with labels for better visibility on grayscale screens.
  - Touch-optimized interface.

## Installation

  1. Download the `kindle-draughts-extension.zip` file.
  2. Unzip it and copy the resulting folder to the `/extensions/` directory on your Kindle.
  3. Launch via **KUAL - > Kindle Draughts -> Launch**.
