# KANKI

> **KAnki** is an Anki-based Kindle flashcard app with spaced repetition, custom decks, and no account requirement.

## Features

  - Spaced repetition review on Kindle
  - Custom decks and language setup
  - Two-way practice and starred cards
  - Progress saved locally on device

## Install

  1. Download and unzip the latest release.
  2. Copy the **KAnki** folder and **kanki.sh** to your Kindle **documents** folder.
  3. Eject the Kindle and launch KAnki from your library.

## Deck Editor

Use the [KAnki Web Editor](https://kindlemodshelf.me/editor.html) to create, import, edit, and preview decks before copying them to Kindle.

## Uben Workflow

  1. Import your `.apkg` deck into [Uben](https://www.uben.online/).
  2. Edit cards in Uben, then export `kanki_config.js`.
  3. Replace your Kindle config file and launch KAnki.

## Notes

  - KAnki is local-first. It does not require an online account.
  - Add a compatible `language.ttf` font for languages that need it.
  - Runs on older Kindle browser tech (ES5 and limited CSS support).
  - Back up your deck/config files before major updates.
  - Community/help: [Discord](https://discord.gg/JtcrpG7ECA)
