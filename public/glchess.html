GlChess

GlChess is a touch-friendly chess application for Kindle, derived from GNOME Chess. It features Cairo-rendered boards and support for UCI engines like Stockfish.

Download

  • Download GlChess Extension (Latest Release)

Features

  • Touch-friendly GTK2 interface optimized for e-ink.
  • Cairo-rendered boards with SVG piece themes.
  • Support for UCI engines (like Stockfish) for AI play.
  • Clean and responsive design.

Installation

  1. Download the kindle-glchess-extension.zip file.
  2. Unzip it and copy the resulting folder to the /extensions/ directory on your Kindle.
  3. Launch via KUAL -> Kindle GlChess -> Launch.
