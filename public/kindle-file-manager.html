Kindle File Manager

Manage Kindle Files with Filebrowser or Syncthing using either KUAL or GUI

Download

  • Download

Prerequisites

  • Jailbroken Kindle
  • Valid wifi network
  • Firmware 5.12.2.2 or newer with Universal Hotfix (Only for Illusion GUI)

Functions and Notes

  • Kindle app using Illusion GUI to start Filebrowser or Syncthing to manage your files.
  • If Illusion is unavailable or KUAL GUI is preferred that is available


Filebrowser is a website that your Kindle hosts to allow you to manage files.

Syncthing syncs all your files from different devices onto your Kindle.
