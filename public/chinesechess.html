Chinese Chess

Chinese Chess (Xiangqi) for Kindle features a standard 9x10 board and supports strong UCI engines like Pikafish for high-level AI play.

Download

  • Download Chinese Chess Extension (Latest Release)

Features

  • Full 9x10 Xiangqi board layout.
  • Integrated support for the Pikafish engine for high-level AI.
  • Built-in fallback AI for casual matches.
  • Touch-friendly board interactions.

Installation

  1. Download the kindle-chinesechess-extension.zip file.
  2. Unzip it and copy the resulting folder to the /extensions/ directory on your Kindle.
  3. Launch via KUAL -> Kindle ChineseChess -> Launch.
