Iagno

Iagno is a port of the Reversi strategy game for Kindle. It features an 8x8 board and multiple AI difficulty levels, optimized for e-ink displays.

Download

  • Download Iagno Extension (Latest Release)

Features

  • Classic 8x8 Reversi gameplay.
  • AI difficulty levels: Easy, Medium, and Hard.
  • Touch-friendly interface designed for Kindle.
  • Crisp rendering optimized for e-ink screens.

Installation

  1. Download the kindle-iagno-extension.zip file.
  2. Unzip it and copy the resulting folder to the /extensions/ directory on your Kindle.
  3. Launch via KUAL -> Kindle Iagno -> Launch.
