# Tetris for KTerm (HF Only)

> **Tetris for KTerm** is a terminal Tetris game for jailbroken HF Kindles, with no emulator required.

## Download

  - [Download kindletetris.zip](../downloads/kindletetris.zip)

## Requirements

  - Jailbroken Kindle with hardfloat (HF) firmware
  - [KTerm terminal](kterm.html) installed

## How to Install

  1. **Download the game:** [Download kindletetris.zip](../downloads/kindletetris.zip)
  2. Unzip the file on your computer.
  3. Copy the extracted files to your Kindle. Place them in your preferred directory (e.g., `/extensions`).
  4. Eject and disconnect your Kindle from your computer.

## How to Play

  1. Open **KTerm** on your Kindle.
  2. Navigate to the directory where you placed the Tetris files using the `cd` command.
  3. Start the game by typing:
`./kindletetris`
  4. Enjoy classic Tetris directly in the terminal window!

## Notes

  - This version works **only** on hardfloat (HF) firmware models. It is not compatible with softfloat (SF) devices.
  - You must launch Tetris from inside KTerm; it is not a standalone app or menu item.
  - Want Game Boy Tetris? Use a Game Boy emulator instead for easier playing.
