# All Kindle Linux Chroots

> Below is a list of known Linux chroot environments ported to the Kindle. Choose the one that best fits your needs: GUI vs. Shell, or specific distro preferences.

## Available Linux Chroots

  - [**Alpine OG (GUI)**](https://github.com/schuhumi/alpine_kindle) — The original Alpine port.
  - [**Alpine Better (newer w/GUI)**](https://github.com/Entersjkhdfkjdhfksjf/alpine_kindle) — An improved version with GUI support.
  - [**Yet another Alpine (shell w/Min GUI)**](https://github.com/ohaiibuzzle/alpine_kindle) — Lightweight shell focused with minimal GUI.
  - [**Debian (GUI)**](https://github.com/Ooonana/Debian_kindle) — Run Debian on your Kindle.
  - [**Fedora (Shell)**](https://github.com/GreenCat-777/fedora_kindle) — Fedora Linux CLI environment.
  - [**Ubuntu (Shell)**](https://github.com/GreenCat-777/ubuntu_kindle) — Ubuntu Linux CLI environment.

## Notes

Instructions are included in the respective repositories for most chroots.

For **Yet another Alpine**, you can find specific instructions [here on the Kindle Modding Discord](https://discord.com/channels/1083603487025274911/1440969313061306409/1440969313061306409).
