# JarLauncher

> Launch jar files on your Kindle, including Minecraft.
>  JarLauncher provides a simple launcher interface for Java applications on jailbroken Kindles.

## Download

[**JarLauncher**](https://github.com/ThatPotatoDev/JarLauncher/releases)

## Prerequisites

  - [KTerm](kterm.html)

## Installation

  1. Download the latest `JarLauncher.zip` from the [releases](https://github.com/ThatPotatoDev/JarLauncher/releases).
  2. Unzip it, then copy the `JarLauncher` folder into the `extensions` folder on your Kindle.
