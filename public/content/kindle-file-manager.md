# Kindle File Manager

> Manage Kindle files with Filebrowser or Syncthing using either KUAL or a GUI interface.

## Download

  - [Download filemanagers.zip](https://github.com/kbarni/kindlefilemanagers/releases/latest/download/filemanagers.zip)

## Features

  - **Filebrowser:** Hosts a local website on your Kindle for managing files via a browser on any connected device
  - **Syncthing:** Syncs files between your Kindle and other devices wirelessly
  - Launch via [Illusion GUI](https://github.com/penguins184/Penguins-Kindle-Wiki/blob/main/Illusion-Guide.md) or KUAL (KUAL available if Illusion is unavailable)

## Prerequisites

  - Jailbroken Kindle
  - Active Wi-Fi connection
  - Firmware 5.12.2.2 or newer with [Universal Hotfix](https://github.com/KindleModding/Hotfix/) (required for Illusion GUI only)
