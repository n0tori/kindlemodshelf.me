# KinAMP

> **KinAMP** is a native music and internet radio player for jailbroken Kindles (FW 5.16+) with KOReader integration.

## Download

[**Download KinAMP**](https://github.com/kbarni/KinAMP/releases/)

## Features

  - Fully native, fast interface (C++ and GTK2)
  - Optimized for e-ink displays to save battery
  - Fast access to Bluetooth and frontlight settings
  - Supports MP3, AAC, FLAC, and WAV formats
  - Internet radio mode (MP3 streams)
  - KOReader plugin — access playlists, songs, and radio from the Tools menu
  - Uses miniaudio for decoding and GStreamer backend

## Installation

  1. Download the latest release from the link above.
  2. Unzip the archive to the root of your Kindle.
  3. Start KinAMP from **KUAL** , the home screen, or via the **KOReader Tools menu**.

## Internet Radio

  - Tap the radio icon in the bottom left corner to switch to radio mode
  - Add or remove stations via the **KinAMP Radio List Editor** in KUAL
  - Currently supports MP3 streams only (no MPEG1/2 or AAC streams)

## Notes

  - **No Background Playback:** This is a GUI app and cannot run in the background. To listen to music while reading, [SOX Media Player](sox.html) is a better alternative.
  - Requires Firmware 5.16 or higher.
